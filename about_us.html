The Technology
Behind WebSafe

Developed by students from STI College Fairview, WebSafe is a website URL malware detector designed to identify and neutralize malicious links. Built on a robust Node.js backend, the system integrates four industry-grade APIs — WHOIS API, Google Safe Browsing, VirusTotal, and Screenshot API — to evaluate URLs in real time and ensure user safety.

Rather than relying on simple blocklists, WebSafe cross-references domain registration data from the WHOIS API, threat intelligence from VirusTotal's 70+ antivirus engines, and Google's own malicious URL database — all evaluated concurrently through Node.js's asynchronous architecture for sub-second results.

A standout component is the Screenshot API-powered preview feature, which renders a live thumbnail of any website inside the WebSafe environment — letting users visually inspect a page without their browser ever touching it, effectively eliminating exposure to auto-downloads and browser exploits.

APIs Used

How WebSafe Is Built

Four industry-grade APIs work together in a multi-layer analysis pipeline — from domain registration data to live threat intelligence and safe visual preview.

01

WHOIS API

Retrieves domain registration data including creation date, registrar, and ownership status. Newly registered or anonymized domains are automatically weighted as higher risk, helping catch freshly spun-up phishing infrastructure.

02

Google Safe Browsing

Cross-references every URL against Google's continuously updated threat lists — covering malware, phishing, unwanted software, and social engineering pages. Flags are returned in real time as part of the safety score calculation.

03

VirusTotal

Submits URLs to VirusTotal's aggregated engine, which checks against 70+ antivirus vendors and URL scanners simultaneously. The consensus result — how many engines flagged the URL — feeds directly into the final safety score.

04

Screenshot API

Renders a live screenshot of the target URL in a sandboxed environment and returns it as an image preview. Users can visually inspect the page's layout and content without their browser ever connecting to the destination server.

What WebSafe Evaluates on Every URL

Each URL is passed through all four integrated APIs concurrently. Results are normalized and weighted — a VirusTotal hit or a Google Safe Browsing flag carries heavier penalties than a domain age warning — and combined into a single safety score from 0 to 100.

The engine weighs each signal differently to ensure the final score accurately reflects the actual threat level rather than treating all checks as equal.

HTTPS & SSL Validation
Verifies end-to-end encryption and certificate validity. Sites without HTTPS lose 30 points immediately — a major red flag for data security.
Google Safe Browsing Lookup
Checks the URL against Google's threat lists covering malware, phishing, and unwanted software — updated in near real-time by Google's crawlers.
VirusTotal Multi-Engine Scan
Submits the URL to 70+ antivirus and URL scanner engines simultaneously. Even a single engine flag triggers a score penalty and an elevated warning.
WHOIS Domain Age & Registrar
Queries registration records to flag newly created or anonymized domains — common tactics used by phishing campaigns to evade detection before blocklists catch up.

Standout Feature

Safe Preview Without Visiting

The most innovative part of WebSafe is its Screenshot API-powered preview — a sandboxed rendering engine captures a live screenshot of the target URL and returns it as a safe image, never exposing the user's browser to the destination.

Combined with VirusTotal and Google Safe Browsing results displayed alongside the preview, users get both a visual impression and a threat intelligence summary before deciding to click.

Built With Purpose

Developed at STI College Fairview

WebSafe was created by students from STI College Fairview as a practical cybersecurity tool for Filipino internet users. Inspired by the rising number of phishing scams targeting local communities, the team integrated WHOIS API, Google Safe Browsing, VirusTotal, and Screenshot API into a free, accessible, no-login-required tool — making professional-grade URL verification simple for everyone.
